  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with the Klef Sonatta website in just a few steps</p>
  
  <Steps>
    <Step title="Clone the Repository">
      Clone the repository to your local machine and navigate to the project directory.
      
      ```bash
      git clone https://github.com/ivanndlezz/klef-sonatta-website.git
      cd klef-sonatta-website
      ```
    </Step>
    
    <Step title="Open in Browser">
      Since this is a static website with no build process, simply open the `index.html` file in your browser.
      
      ```bash
      # For a better development experience, use a local server
      python -m http.server 8000
      ```
      
      <Info>The website works without a server, but using one provides a better development experience and proper file path resolution.</Info>
    </Step>
    
    <Step title="Explore the Components">
      Navigate through the site to see the mega menu, portfolio grid, and dynamic components in action. Check out the [Components documentation](/components/mega-menu) for detailed implementation details.
    </Step>
  </Steps>
</div>
